# health_tracker

A Flask-based healthcare assistant chatbot that provides general wellness information, diet planning, and daily health tips.
Quick Start
Prerequisites
Python 3.7+
pip

Installation & Setup

Create project structure:

mkdir healthcare-chatbot
cd healthcare-chatbot
mkdir templates

Save the files:

Save app.py in the root directory
Save index.html in the templates folder


Install Flask:

bashpip install flask

Run the application:

bashpython app.py
Or:
bashflask run

Open in browser:
Navigate to http://127.0.0.1:5000/
